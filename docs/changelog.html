0.8.0

c-3po lib

  • Deleted setDefaultHeaders method. You should call new setDefaultLang to change the default language.
  • removed c-3po/loader and gettext-parser dependency, beacause it is little bit out of the scope of the library. You should import and call parser separately.
import gt from 'gettext-parser';
import fs from 'fs';

gt.po.parse(fs.readFileSync('file.po'));

c-3po-plugin

  • Deleted defaultHeaders setting from the config. You should use defaultLang instead.
  • Added new setting numberedExpressions to allow any expressions inside translated strings.
